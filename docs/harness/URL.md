# Harness Knowledge URL Map

Website hiện dùng một route chính `/` theo phong cách documentation-driven website, với các mục nội dung đi bằng anchor để giữ trải nghiệm đọc liền mạch.

| Mục chính | URL |
| --- | --- |
| Trang chủ | `/#trang-chu` |
| Khái niệm | `/#khai-niem` |
| Nguyên lý | `/#nguyen-ly` |
| Dự án | `/#du-an` |
| Tài nguyên | `/#tai-nguyen` |
| Về chúng tôi | `/#ve-chung-toi` |

Khi nội dung đủ lớn, có thể tách các anchor này thành route riêng như `/khai-niem`, `/nguyen-ly`, `/du-an`, `/tai-nguyen`, `/ve-chung-toi` mà vẫn giữ cấu trúc điều hướng hiện tại.
